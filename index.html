Help

How to retrieve a POD

  1. Input either:
    • Order Number (i.e. “583WS”)
    • Consignment Number (TNT “Shipment Number” i.e. “71690149”)
    • Return Number (i.e. “R6611446”)
  2. Click “Get POD” or hit Enter.

Controls

  • Account Number – Current TNT account number. Click to change/update. NOTE: No retrieve will be possible if this is incorrect.
  • 📌Pin Button – Pin the selected POD and add a comment note so it will be retrieved whenever the page is loaded. Click again to unpin.
  • POD image – Double click to view in a new window.
  • Tracking Button – Click to view tracking on TNT.com
  • ✉Email Button – Click to open a pre-filled email to sales@. NOTE: You will need to manually copy and paste the actual POD image into the email.
  • Auto check Button – Click to automaticaly check for updates every couple minutes and notify you when a change occurs.

Troubleshooting

If no record is loaded when you search, check:
  1. Order, Consignment, or Return number is correct (must be a TNT delivery of course) and released for delivery.
  2. TNT.com is up and running – Sometimes all records are temporarily unavailable, if so just wait a bit and try again.
